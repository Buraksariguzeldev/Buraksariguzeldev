Termux'ta PHP Kurulumu

Termux üzerinde PHP'yi kurmak için aşağıdaki adımları takip edin:

  • Termux uygulamasını açın.
  • Aşağıdaki komutu kullanarak PHP paketini yükleyin:
    pkg install php
  • Kurulum tamamlandıktan sonra, PHP'nin başarıyla yüklendiğini kontrol etmek için şu komutu girin:
    php -v
  • Artık PHP yüklendi, bir PHP betiği oluşturabilir ve çalıştırabilirsiniz.

Basit Bir PHP Betiği Oluşturma ve Localhost'ta Çalıştırma

Bir PHP dosyası oluşturmak ve yerel bir sunucu üzerinde çalıştırmak için:

  • Termux'ta nano veya başka bir metin editörünü kullanarak yeni bir PHP dosyası oluşturun, örneğin:
    nano test.php
  • Aşağıdaki içeriği dosyaya yazın ve kaydedin:
    <?php\n echo "Merhaba, PHP!";\n?>
  • PHP yerel sunucusunu başlatmak için aşağıdaki komutu kullanın:
    php -S localhost:8080
  • Herhangi bir web tarayıcısında, şu adresi ziyaret edin:
    localhost:8080/test.php Bu adım, PHP yerel sunucusu üzerinde test.php dosyanızı yükleyecektir.
