Termux'ta PHP Kurulumu

PHP'nin Termux Üzerinde Kurulumu

Termux üzerinde PHP'yi kurmak için aşağıdaki adımları takip edin:

  1. Termux uygulamasını açın.
  2. Aşağıdaki komutu kullanarak PHP paketini yükleyin: 
    pkg install php
  3. Kurulum tamamlandıktan sonra, PHP'nin başarıyla yüklendiğini kontrol etmek için şu komutu girin: 
    php -v

Basit Bir PHP Betiği Oluşturma ve Localhost'ta Çalıştırma

Bir PHP dosyası oluşturmak ve yerel bir sunucu üzerinde çalıştırmak için:

  1. Termux'ta `nano` veya başka bir metin editörünü kullanarak yeni bir PHP dosyası oluşturun, örneğin: 
    nano test.php
  2. Aşağıdaki içeriği dosyaya yazın ve kaydedin: 
    <?php\n    echo "Merhaba, PHP!";\n?>
  3. PHP yerel sunucusunu başlatmak için aşağıdaki komutu kullanın: 
    php -S localhost:8080
  4. Herhangi bir web tarayıcısında, şu adresi ziyaret edin: 
    localhost:8080/test.php
    Bu adım, PHP yerel sunucusu üzerinde `test.php` dosyanızı yükleyecektir.
