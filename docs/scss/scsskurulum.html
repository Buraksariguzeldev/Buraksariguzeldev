Sass Otomatik Derleme

Sass Kurulumu

Sass'ı Termux'ta kullanmak için aşağıdaki adımları izleyebilirsiniz:

  • Node.js ve npm Kurulumu: Sass'ı kullanmadan önce, Termux'ta Node.js ve npm'in kurulu olması gerekmektedir. Eğer henüz yüklü değilse, aşağıdaki komutu kullanarak kurabilirsiniz:
    pkg install nodejs
  • Sass Kurulumu: Node.js ve npm kurulduktan sonra, Sass'ı yüklemek için npm'i kullanabilirsiniz. Aşağıdaki komutu kullanarak Sass'ı global olarak yükleyebilirsiniz:
    npm install -g sass

Otomatik Derleme

  • Eğer Sass dosyaları ve CSS dosyaları aynı dizindeyse, otomatik derleme için Termux'ta aşağıdaki komutu kullanabilirsiniz:
    sass --watch dosya_adı.scss:dosya_adı.css

Burada, dosya_adı.scss Sass dosyasının adını ve uzantısını temsil ederken, dosya_adı.css ise derlenmiş CSS dosyasının adını ve uzantısını belirtir. Bu komut, dosya_adı.scss dosyasındaki herhangi bir değişikliği izleyecek ve her değişiklik algılandığında otomatik olarak dosya_adı.css dosyasını güncelleyecektir.

SCSS Temel Özellikleri

SCSS, CSS'in geliştirilmiş bir sürümüdür ve birçok kullanışlı özellik sunar:

  • Değişkenler: Renkler, fontlar ve diğer değerler için değişkenler tanımlayabilirsiniz.
  • Nesting: CSS kurallarını daha düzenli bir şekilde organize etmek için nesting kullanabilirsiniz.
  • Mixinler: Tekrar eden CSS kodlarını fonksiyon gibi kullanarak mixinlerle birleştirebilirsiniz.
  • Extend: Bir CSS kuralını başka bir kurala genişletebilir ve yeniden kullanabilirsiniz.

Bunlar sadece SCSS'in temel özelliklerinden bazılarıdır. Daha fazla bilgi için SCSS dökümantasyonuna göz atabilirsiniz.
