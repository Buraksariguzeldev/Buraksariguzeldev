SCSS Kullanımı

Temel SCSS Komutları

SCSS kurulumu ve kullanımı ile ilgili temel komutlar:

1. SCSS Kurulumu

npm install -g sass

2. SCSS Dosyasını CSS'e Dönüştürme

sass sasskursu.scss sasskursu.css

3. SCSS İzleme Modu

Bu mod, SCSS dosyalarındaki değişiklikleri otomatik olarak algılar ve CSS'e dönüştürür.

sass --watch sasskursu.scss:sasskursu.css

SCSS Dosyası Örneği

Bir SCSS dosyasında kullanılan temel yapı:

    \$primary-color: #333;
    \$font-stack: Helvetica, Arial, sans-serif;

    body {
        font-family: \$font-stack;
        color: \$primary-color;
    }

    h1 {
        color: darken(\$primary-color, 20%);
    }

Ekstra SCSS Özellikleri

SCSS ile kullanabileceğiniz bazı ekstra özellikler:

  • İç içe Yazım: SCSS, CSS seçicilerini iç içe yazmanıza olanak tanır, bu da kodunuzu daha düzenli ve okunabilir kılar.
  • Mixins: Tekrar eden CSS bloklarını yeniden kullanmak için mixins oluşturabilirsiniz.
  • Extend/Inheritance: Ortak stil özelliklerini paylaşmak için, bir seçiciyi başka bir seçiciye genişletebilirsiniz.
  • Fonksiyonlar: Renkleri ayarlamak, matematiksel hesaplamalar yapmak gibi işlemler için SCSS fonksiyonlarını kullanabilirsiniz.
