Termux'ta Python ile Basit HTTP Sunucusu Kurulumu

Python Yükleme

Termux'a Python yüklemek için:

pkg install python

HTML Dosyası Oluşturma

Bir HTML dosyası oluşturun:

echo "<h1>Merhaba Dünya!</h1>" > index.html

Python HTTP Sunucusunu Başlatma

Python ile basit bir HTTP sunucusu başlatmak için:

python -m http.server 8080

Tarayıcıda Erişim

Web tarayıcınızda `localhost:8080` adresine giderek HTML dosyasını görüntüleyin.
