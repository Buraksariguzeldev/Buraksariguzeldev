Python Kurulumu

Python'u Termux'ta kurmak için aşağıdaki adımları izleyin:

  • Termux uygulamasını açın.
  • Aşağıdaki komutu kullanarak Python paketini yükleyin:
    pkg install python
  • Python sürümünü kontrol edin:
    python --version

Python başarıyla kurulduysa, bu komut Python sürüm numarasını gösterecektir.

Python HTTP Sunucusu Kurulumu ve Kullanımı

Python HTTP sunucusunu başlatmak için aşağıdaki adımları izleyin:

  • Mevcut dizine gidin:
    cd /storage/emulated/0/Buraksariguzeldev
  • Python HTTP sunucusunu başlatmak için aşağıdaki komutu kullanın:
    python -m http.server

Bu komut, mevcut dizinde bulunan dosyaları sunmak için bir HTTP sunucusu başlatacaktır. Tarayıcınızda http://[::]:8000/%C4%B0ndex.html adresine giderek sunucunuzu görebilirsiniz.

Python Hakkında

Python, genel amaçlı, yorumlanabilir, yüksek seviyeli bir programlama dilidir. Kullanım alanları arasında web geliştirme, veri analizi, yapay zeka, bilimsel hesaplama ve otomasyon bulunur.

Python'un temel özellikleri arasında kolay okunabilirlik, açık ve anlaşılır sözdizimi, geniş bir standart kütüphane ve topluluk desteği yer alır.

Python, hem başlangıç düzeyindeki kullanıcılar hem de deneyimli programcılar için ideal bir seçenektir.
