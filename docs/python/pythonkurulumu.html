Termux Python ve Localhost Kurulumu

Python Kurulumu

Python'u Termux'ta kurmak için aşağıdaki adımları izleyin:

  • Termux uygulamasını açın.
  • Python paketini yüklemek için şu komutu kullanın:
    • pkg install python
  • Python sürümünü kontrol edin:
    • python --version

Eğer Python başarıyla yüklendiyse, python --version komutu Python sürüm numarasını gösterecektir.

Localhost Üzerinde Python Sunucusu Başlatma

Localhost'ta bir Python sunucusu başlatmak için aşağıdaki adımları izleyin:

  • Dosyalarınızı içeren dizine gidin:
    • cd /dizin/yolu
  • Python HTTP sunucusunu başlatmak için şu komutu kullanın:
    • python -m http.server

Bu komut, dosyalarınızı localhost'ta bir HTTP sunucusu üzerinde yayınlayacaktır. Tarayıcınızda http://localhost:8000 adresine giderek sunucunuzu görebilirsiniz.
